# Contributing to HG PPC

## Welcome

We welcome contributions to the HG PPC project! This document provides guidelines for contributing to the project.

## Getting Started

1. Fork the repository
2. Clone your fork
3. Create a new branch for your feature or bugfix
4. Make your changes
5. Run tests and linting
6. Submit a pull request

## Development Setup

1. Install dependencies:
   ```bash
   pnpm install
   ```

2. Set up local environment:
   - Copy `.env.example` to `.env.local`
   - Fill in required configuration

## Development Workflow

- Use feature branches
- Commit messages should be clear and descriptive
- Run `pnpm lint` before committing
- Ensure all tests pass

## Pull Request Process

1. Ensure your code follows project standards
2. Update documentation as needed
3. Your pull request will be reviewed by maintainers

## Code of Conduct

- Be respectful and inclusive
- Collaborate constructively
- Report any inappropriate behavior

## Questions?

If you have questions, please open an issue or contact project maintainers.
